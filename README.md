# 🌤 AI Weather & Forecast Bot

A smart Telegram bot that provides real-time weather, wind data, and 3-day forecasts with a touch of AI humor.

## ✨ Features
* **Global Weather:** Get current conditions for any city worldwide.
* **Wind Info:** Dedicated command to check wind speed.
* **3-Day Forecast:** Detailed forecast including min/max and average temperatures.
* **Persistent Settings:** Remembers your last searched city using **SQLite**.
* **AI Quotes:** Generates funny "meme" quotes based on the temperature using **Groq (Llama 3.3)**.

## 🛠 Tech Stack
* **Language:** Python
* **API:** [wttr.in](https://wttr.in) (Weather data)
* **Database:** SQLite3
* **LLM:** Groq Cloud API (Llama 3.3 70B)
* **Framework:** PyTelegramBotAPI (Telebot)
